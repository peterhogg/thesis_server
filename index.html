Supervisors:
Randy Fortier, and Ken Pu

Introduction

Designed for lecture environment
Slides send information to android application in real time
Students give responses through application

Justification

There is useful data to be collected during a lecture This facilitates the collection of this data Gives users a way to directly send feedback during presentation Professors can respond immediately to the feedback sent back to them ie if a lot of people liked the topic, get into more detail, or provide additional resources. If the understand rating of the slide is low, go into more detail

Similar Product

Top Hat (www.tophat.com)
“Top Hat is a comprehensive teaching platform that helps professors like you easily create an interactive lecture experience.”
Tophat is a similar lecture companion platform Tophat is based off of a subscription model, and the code is proprietary

Features

Sends users the title, and description of the current topic Gives users the option to like a topic Provides a slider for users to respond with how well they’ve understood the topic. Display this information directly on the slide.

Like Button

The like button allows users to like a certain topic. The slide calculates how many users liked that topic

Understand Bar

User’s can select how well they understood a topic Customizable max and min to fit needs of the professor Slides will report on the average understanding

Benefits of Real Time Classroom Analytics

Software is open sourced
Hooks into revealJS
Delivers information in real time

Technology Stack

Phone Client(Android)
Real Time Communication(Socket IO)
Server (Node JS, Express)
Slide Show(Reveal JS)

Android

RevealJS

RevealJS allows for the creation of presentations using HTML. All the slides available online
Visual Editors available (slides.com)
Easy to plug into through javascript

Custom XML Fragment

SocketIO

Socket IO is a socket library allowing bidirectional data flow Has great in browser support, powers the slides sending the data to the phone application. Great native android support as well

NodeJS

Node, using Express serves the slides Node also acts as a dispatcher for the socket connections Assignes a UUID to each client as it connects Facilitates sending messages from slide, to android clients Stores like, and understand information
